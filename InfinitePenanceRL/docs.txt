Описание компонентов игры:

Camera.cs
- Отвечает за отслеживание видимой области игрового мира
- Следует за игроком и обеспечивает плавную прокрутку
- Преобразует мировые координаты в экранные

ColliderComponent.cs
- Компонент для обработки столкновений
- Определяет границы объектов для физического взаимодействия
- Проверяет пересечения с другими объектами

Component.cs
- Базовый класс для всех игровых компонентов
- Обеспечивает связь с сущностью-владельцем
- Предоставляет доступ к игровому движку

Entity.cs
- Базовый класс для всех игровых объектов
- Содержит позицию и список компонентов
- Управляет жизненным циклом компонентов

EntityFactory.cs
- Создает различные типы игровых объектов
- Настраивает компоненты для каждого типа объекта
- Инициализирует начальные параметры

GameEngine.cs
- Основной класс игрового движка
- Управляет всеми системами (рендеринг, физика, ввод)
- Координирует работу всех компонентов

HealthBarComponent.cs
- Отображает полоску здоровья
- Визуализирует текущее и максимальное здоровье
- Обновляется при изменении здоровья

InputManager.cs
- Обрабатывает пользовательский ввод
- Отслеживает нажатия клавиш
- Предоставляет информацию о состоянии ввода

InventoryComponent.cs
- Управляет инвентарем игрока
- Отображает предметы и их количество
- Обрабатывает взаимодействие с предметами

MainForm.cs
- Главное окно приложения
- Обрабатывает события Windows Forms
- Управляет игровым циклом

MovementComponent.cs
- Отвечает за перемещение объектов
- Обрабатывает пользовательский ввод для движения
- Применяет физические ограничения

PhysicsSystem.cs
- Обрабатывает физические взаимодействия
- Проверяет возможность перемещения
- Разрешает коллизии между объектами

RenderComponent.cs
- Отвечает за визуализацию объекта
- Управляет спрайтами и их отрисовкой
- Поддерживает различные режимы отображения

RenderSystem.cs
- Система отрисовки всех объектов
- Управляет порядком отрисовки
- Применяет преобразования камеры

Scene.cs
- Представляет игровой уровень
- Содержит все объекты на сцене
- Управляет жизненным циклом объектов

SpriteManager.cs
- Загружает и управляет спрайтами
- Хранит области спрайтов на спрайтшитах
- Отрисовывает спрайты с нужным масштабом

UIComponent.cs
- Базовый класс для элементов интерфейса
- Обеспечивает позиционирование на экране
- Управляет видимостью элементов

UIManager.cs
- Управляет всеми элементами интерфейса
- Координирует отображение UI компонентов
- Обрабатывает взаимодействие с интерфейсом

Vector2.cs
- Структура для работы с 2D векторами
- Используется для позиций и перемещений
- Поддерживает векторные операции
